# Bluewser
Новый apk-браузер для Вашего телефона.
В Plugins устанавливаете Javascript-код(ы) на свой страх и риск! Внутри Plugins нет поддержки HTML.
Если не работает скролл в браузере (зависит от Android), то копируете scroll.js, вставив в Plugins этот js-код.

## Если хотите дополнить функционал браузера, то можете попросить у ИИ любой Javascript-код, например "темную тему" или другой функционал.
